# Portfólio – Antonio Luiz Santos

Portfólio pessoal desenvolvido com HTML, CSS e JavaScript, apresentando projetos, habilidades técnicas e formas de contato.

## Tecnologias

- HTML5
- CSS3
- JavaScript
- React

## Funcionalidades

- Carrossel de projetos com loop infinito e pausa no hover
- Tooltips nas tech cards com descrição de cada tecnologia
- Lazy loading com skeleton nas imagens dos projetos
- Scroll spy com destaque automático no menu conforme a seção ativa
- Scroll suave com offset do header fixo
- Tema claro e escuro com preferência salva no localStorage
- Menu hambúrguer responsivo para mobile
- Animações de reveal e parallax nas seções
- Logo personalizada no header com link para o topo
- Links de projetos com aria-label para acessibilidade
- Meta description para SEO básico
